# Scenario B — 한국무용 회상 인지중재 모드

> **Status**: Secondary, **Phase 0 즉시 상용화 핵심**
> **Source**: 통합의학 석사 + 따님 한국무용 자산 + 차움 회원전담 5년의 정서·콘텐츠 큐레이션 노하우
> **Regulatory target**: 의료기기 비해당 (건강관리 분류) — 6개월 내 시장 출시 가능

## 1. 왜 이 시나리오가 Phase 0 메인인가

치매·MCI 환자 350만 명. 비약물 인지중재(Cognitive Stimulation Therapy, CST)의 효과는 *Lancet Psychiatry 2020*에서 효과크기 0.41로 입증 — 도네페질(아리셉트)의 0.20보다 크다. 그러나 한국 가정에 *효과적·문화적합한* CST 콘텐츠는 부재.

기존 시장의 빈틈:
- **효돌이류**: 단순 안부 인사 + 노래방, CST 효과 미입증
- **외산 콘텐츠 (CogniFit 등)**: 영어/서양 문화권, 한국 노인 비친화
- **치매안심센터 프로그램**: 주 1회 방문, 연속성 부족

삼온의 차별 두 가지:
1. **콘텐츠 IP의 한국 정체성** — 한국무용·국악·세대별 회상 사진(60·70년대 결혼식, 80년대 가요), 가요무대·인간극장 명장면 — *대표님 따님 한국무용 + 통합의학 + VIP 큐레이션 = 노희영식 K-콘텐츠*
2. **개인 메모리 기반 큐레이션** — 환자가 *어떤* 콘텐츠에 어떻게 반응하는지 학습해서 다음 회기 조정 (Layer 5 자기반성)

## 2. User persona

| Field | Value |
|-------|-------|
| 이름(가명) | 박순자 |
| 나이 | 76세 여성 |
| 진단 | MCI (경도인지장애), MMSE 25점 |
| 거주 | 며느리·아들과 동거 (낮 시간 혼자) |
| 배경 | 1971년 결혼, 1973년 첫째 출산, 자영업(미용실) 30년, 2년 전 폐업 |
| 정서 | 폐업 + 남편 사별(2024) 후 *"할 일이 없다"* 호소, 우울 점수 0.5 |
| 음악 선호 | 이미자 *동백 아가씨*, 패티김 *이별*, 트로트 일반 |
| 신체 | 고혈압, 무릎 관절염 — 외출 어려움 |

## 3. End-to-End Flow

### 3.1 Trigger
오후 3시 12분, 환자가 거실 소파에서 30분간 무응답·무활동.

### 3.2 Layer-by-layer

**Layer 1 (Perception)**
- Vision: 표정 분석 → 무표정 + 시선 허공
- Audio: 무음
- Wearable: HR 65 (안정), 활동량 30분 0
- Spatial: 소파 위치 고정

**Layer 2 (Plan)**
- 컨텍스트: `time=15:12`, `inactivity=30min`, `mood_baseline=mild_depression`
- 모드 결정: `cognitive_intervention_invitation`
- 목표: 비강요 방식으로 콘텐츠 제안

**Layer 3 (Skill)**
```
robot.move_to("소파_정면_1m")
robot.gaze_at(patient.face)
robot.say("순자님, 오늘 햇볕이 참 좋네요. 제가 사진 한 장 보여드려도 될까요?")
# 환자 동의 후
robot.play_content("wedding_photos_1971_korean")
```

**Layer 4 (Memory)**
회상:
- *"순자님은 1971년 결혼식 사진을 보여드리면 30분간 차분해진 패턴이 5회 누적"*
- *"미용실 운영 시절(1990년대 종로) 이야기를 시작하면 30~45분간 활발해진다"*
- *"이미자 동백 아가씨에 가장 강한 정서 반응"*

이 회상이 콘텐츠 큐레이션을 결정.

**Layer 5 (Reflection — async)**
세션 종료 후 분석:
- 시청 시간 / 표정 변화 / 발화 빈도
- 가족 피드백 (며느리: *"오늘 어머님 점심 잘 드셨어요"*)
- 다음 세션 콘텐츠 우선순위 자동 조정

### 3.3 Conversation flow

```
ROBOT: 순자님, 오늘 햇볕이 참 좋네요. 사진 한 장 보여드려도 될까요?
       (디스플레이: 1971년 결혼식 사진 thumbnail)

박순자: (잠시 보더니) ...어머, 이거 우리 결혼식이네.

ROBOT: 네 맞아요. 1971년 5월 6일이라고 들었어요.
       그날 어떤 분들이 오셨는지 기억나세요?
       (전체 화면: 결혼식 사진)

박순자: 우리 어머니는 남색 한복 입으셨고... 시어머니는 분홍이었어.
       작은오빠가 함진아비 했고...

ROBOT: 함진아비 하실 때 무슨 노래 부르셨어요?

박순자: 그때 다들 이미자 노래 좋아했지. 동백 아가씨...

ROBOT: 동백 아가씨 한번 같이 들어볼까요?
       (재생 — 음량 적당히, 자막 큰 글씨로)

박순자: (따라 부르며) 헤일라 헤일라 ...
```

응답 설계 원칙:
- **개방형 질문 + 환자 발화 인용 반복** — 인지자극 + 자존감 유지
- **사진 → 음악 → 회상 대화의 30분 아크 구성** — CST 표준 모델
- **신체 한계 고려** — 일어서기 권유 절대 금지(관절염), 손동작·시선·박자 맞추기 위주

### 3.4 한국무용 통합 (대표님 따님 자산 활용)

주 2회 *"한국무용 함께 보기"* 프로그램:
- 콘텐츠: 살풀이춤(슬픔·치유), 강강술래(공동체·즐거움), 부채춤(우아·자긍심)
- 환자가 보는 동안 손동작·박자 따라하기 유도 (가능한 범위)
- 따님과 협업으로 *어르신 친화 한국무용 시리즈* IP 확보 가능 — 글로벌 K-컬처 수출 자산

## 4. Demo plan (PoC)

TurtleSim 환경에 매핑:

| Step | Visualization |
|------|---------------|
| 1 | 환자 무응답 30분 → 시스템 트리거 |
| 2 | 메모리 회상 (1971 결혼식 + 동백 아가씨 강조) |
| 3 | 콘텐츠 제안 → 환자 동의 |
| 4 | 회상 대화 시뮬레이션 텍스트 |
| 5 | 세션 후 자기반성 — 다음 세션 콘텐츠 큐 |

데모 시간: 1분 30초.

## 5. KPI (Phase 0)

- 환자 세션 평균 참여 시간 20분+ (대조군 대비 ×2)
- MMSE 점수 6개월 후 0.5점 상승 또는 유지 (대조군 평균 0.7점 하락)
- 가족 NPS 60+
- 콘텐츠 좋아요 / 다시보기 비율 40%+

## 6. Business value & 글로벌 확장

**Phase 0 (3개월) — 즉시 출시 가능**: 의료기기 비해당 분류로 인증 절차 없음. B2C 250만 원 + 월 5만 원 구독.

**글로벌 IP 자산**:
- 일본: 고령화율 29%, 한류 + 트로트 시장 정확히 일치
- 동남아: 한국 콘텐츠 친숙층 (베트남·태국·인도네시아 60대+)
- 중동: K-헬스케어 의료관광과 결합

콘텐츠는 *"몸의 온기는 약으로, 마음의 온기는 예술로"* 라는 노희영식 카피로 포지셔닝.

## 7. 안전성 가드레일

- **자존감 보호 발화 원칙**: 환자의 회상이 부정확해도 *교정 금지*. 회상은 정확성이 아니라 정서 반응이 목적.
- **세션 강제 종료 금지**: 환자가 *"그만"* 또는 무관심 신호 → 즉시 부드럽게 종료
- **가족 영상 공유는 별도 동의** — 메모리에 저장된 가족사진은 환자 본인 외 표시 금지
- **우울·자살 신호 감지 시** 인지중재 즉시 중단 → 가족·의료진 알림으로 라우팅
