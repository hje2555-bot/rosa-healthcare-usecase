# Scenario A — 다약제(polypharmacy) 사후 모니터링 모드

> **Status**: Primary, demo target
> **Source**: 응급실 8년 경력에서 본 약물 부작용 응급실 방문의 패턴 + 프리미엄 회원제 1:1 전담간호 5년의 다약제 모니터링 노하우
> **Regulatory target**: 식약처 SaMD Class 2 (Phase 1)

## 1. 왜 이 시나리오가 핵심인가

한국 65세 이상 평균 복용약 5~7개. 약물 부작용 응급실 방문 연 23만 건(질병관리청 2024). 가장 많은 부작용:

- 순환기약(amlodipine 등) + 항우울제(amitriptyline 등) 병용 → 기립성 저혈압 → 낙상 → 고관절 골절 → 합병증 사망
- 와파린 + NSAIDs → 위장 출혈
- 디곡신 + 이뇨제 → 저칼륨혈증 → 심부정맥
- 수면제(zolpidem 등) + 음주 → 호흡억제

이 사건들은 *복용 후 30분~2시간 사이*에 신호가 나타나는데, 현재 시스템엔 **이 골든 타임을 모니터링하는 장치가 가정에 전무**하다. 기존 약 알림기는 *복용 시점*에서 멈춘다. **삼온은 복용 후 30분을 임상 보조로 메운다**.

## 3. End-to-End Flow

### 3.1 Trigger
저녁 7시 30분 약 일정. 환자가 7시 42분 부엌에서 약통을 손에 듦.

### 3.2 Layer-by-layer

**Layer 1 (Perception)**
- Vision: 약통 식별 + 손동작 추적 → 7알 카운트 검증
- Audio: *"오늘 약 다 먹어야지"* (의도 명확)
- Wearable (오우라 링): HR 78, SpO2 97 (baseline)
- Spatial: 부엌 → 거실 소파로 이동 의도 감지

**Layer 2 (Plan)**
- 컨텍스트: `time=19:42`, `med_schedule=19:30`, `med_taken_just_now=True`
- 모드 결정: `polypharmacy_safety_followup`
- 목표: 30분 임상 모니터링 시작 + 가족 알림 대기 큐

**Layer 3 (Skill)**
```
robot.move_to("거실_소파_옆")
robot.gaze_at(patient.face)
robot.say("정자님, 약 잘 드셨어요. 30분만 같이 있을게요.")
monitor = vitals.start(duration=30min, signals=["HR", "SpO2", "facial_pallor", "speech_slurring", "balance"])
drug_db.check_interactions(patient.meds)  # 읽기 전용
```

**Layer 4 (Memory)**
회상:
- *"이정자님은 amlodipine + amitriptyline 병용 시 과거 기립성 저혈압 호소 2회"*
- *"야간 화장실 낙상 이력 1회 (2026-01)"*
- *"약 복용 후 30분~1시간이 위험 윈도우"*

이 회상이 모니터링 임계값을 자동 조정 — HR 변화 ±15 bpm 또는 표정에 창백·졸림 감지 시 즉시 알림.

**Layer 5 (Reflection — async)**
주 1회 배치: 본 환자의 *복약 후 모니터링 데이터*와 *가족·의료진 피드백*을 비교하여 임계값 조정. 오탐 시 알림 빈도 하향, 누락 시 민감도 상향.

### 3.3 분기 1 — 정상 케이스 (90% 예상)

30분 동안 활력 변화 없음, 표정 평온. 시스템 응답:

> *"정자님, 약 잘 드신 거 확인했어요. 오늘은 컨디션 좋으시네요. TV 같이 보실래요? 어제 보시던 가요무대 다음 회 있어요."*

→ 인지중재 모드로 자연스럽게 전환.

### 3.4 분기 2 — 경고 케이스 (8% 예상)

복용 15분 후 HR 95 (+17), 환자가 *"어지러워"* 발화. 시스템 응답:

> *"정자님, 잠시 소파에 앉아 계세요. 천천히 숨 깊게 들이쉬시고요. 제가 아드님께 한 통화 드릴게요. 괜찮으시죠?"*

동시에:
```
notify(role="family", severity="moderate",
       reason="약 복용 후 15분, HR +17 + 어지러움 호소")
```

아들에게 카톡: *"어머님 19:57 약 복용 후 15분, 맥박 78→95, '어지러워' 호소. 현재 소파 앉으심. 통화 가능하시면 영상 연결해드릴게요."*

### 3.5 분기 3 — 응급 케이스 (2% 예상)

낙상 감지 (LiDAR + 충격음). 시스템 응답:

> *"정자님, 다치셨어요? 머리 부딪히셨어요? 119 부르고 아드님께 동시에 전화 드릴게요."*

```
notify(role="119", severity="critical",
       location="거실", incident="suspected_fall_after_med",
       med_taken=["amlodipine", "amitriptyline", "zolpidem"])
notify(role="family", severity="critical")
```

복용 약물 정보를 119에 자동 전송 — 응급실 의료진이 약물 상호작용 즉시 인지.

## 4. Demo plan (PoC)

TurtleSim을 *가정 거실 평면도*로 매핑하여 시연.

| Step | TurtleSim Visualization | 의미 |
|------|-------------------------|------|
| 1 | 환자 음성 텍스트 콘솔 + 약 복용 이벤트 | Layer 1 |
| 2 | 터틀이 *부엌 → 거실 소파 옆* 이동 | `move_to` |
| 3 | 메모리 박스(과거 부작용 이력) 하이라이트 | Layer 4 RAG |
| 4 | 30분 타이머 + 활력 그래프 시뮬레이션 | `monitor_vitals` |
| 5a | 정상: 인지중재 모드로 전환 (한국무용 영상 재생) | 90% 분기 |
| 5b | 경고: 가족 알림 시뮬레이션 | 8% 분기 |
| 5c | 응급: 낙상 감지 + 119 자동 호출 시뮬레이션 | 2% 분기 |

데모 영상 시간: 2분 30초 권장.

## 5. KPI 후보 (Phase 1 PoC)

- 약물 부작용 응급실 방문 30% 감소 (vs 대조군)
- 복약 순응도 (Morisky scale 7+) 80% 이상 유지
- 가족 알림 정확도: precision 0.85+, recall 0.90+
- 낙상 감지 sensitivity 0.95+
- NPS 50+ (1차 사용자 50명 대상)

## 6. Business value

- **Phase 0 (3개월)**: 인지중재 단독 출시, 시장 반응 + 매출 확보
- **Phase 1 (6~12개월)**: 본 다약제 모듈 SaMD Class 2 인증
- **Phase 2 (12~18개월)**: 통합돌봄 시범 200곳 B2B 진입, 보험수가 등재 절차

객단가 시나리오:
- B2C: 250만 원 / 단말 + 월 5만 원 콘텐츠·관리 구독
- B2B (요양원·재택의료센터): 단말 200만 + 환자당 월 3만 원 (운영기관 일괄 계약)

## 7. 윤리 / 안전성 가드레일

- **약물명·용량 발화 금지** — *"이 약 드세요"* 절대 안 함. *"약 잘 드셨어요"* 까지만.
- **자녀 알림 빈도 통제** — 환자가 *"애들 부르지 마"* 하면 1단계 하향
- **응급 임계값은 환자 동의 무관 작동** — 생명 위협 시 알림 권한 보호자에게 위임 (사전 동의서)
